# n-puzzle
A React n-puzzle game

## Getting started
- ```git clone https://github.com/fn753/n-puzzle.git ```
- ```cd n-puzzle```
- ```npm install```
- ```npm start```

## Change the field size
- Edit src/App.js
- Change value "col" (line 11) for number of columns
- Change value "row" (line 12) for number of rows
